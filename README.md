# TrailcamPhotosToWebsite

Webapp for watching trailcam images with Node.js backend and react frontend.

## Prerequisites
```
Node.js 
```

## Installation

### Backend

Install required depencies:
```
npm install 
```

Run server at localhost:5000:
```
npm start 
```

### Frontend

Install required depencies:
```
cd client
npm install
```

Run to start react-frontend at localhost:3000:
```
cd client
npm start 
```
